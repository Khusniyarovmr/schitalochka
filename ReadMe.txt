Это интерактивная копия моей финансовой таблицы.

Функция отображения с настройками изменения количества периодов.
Функция добавления статей доходов и расходов и редактирования сумм для каждого периода.
Зарплатные таблички тоже должны присутствовать.

Дальнейшее развитие получит в качестве инвестиционного портфеля.

Для сервера будем использовать django \ Flask
Для клиентской части на компе - PyQt5\6
Для клиентской части на смартфоне - KiVi
Для получения данных вероятно requests
Возможность выгружать в Excel \ google docks
Расчеты отпускных и после отпускных выплат
